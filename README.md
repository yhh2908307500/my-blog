# My Blog

Welcome to my personal blog system built with [Hugo](https://gohugo.io/) and deployed on [GitHub Pages](https://pages.github.com/). This blog uses the [hugo-book](https://github.com/alex-shpak/hugo-book) theme for a clean and documentation-style layout.

- **Source Code**: [GitHub Repository](https://github.com/yhh2908307500/superking-blog)
- **Preview**: [https://yhh2908307500.github.io/superking-blog/](https://yhh2908307500.github.io/superking-blog/)

## Project Background

This project serves as a personal knowledge base and sharing platform. It leverages Hugo's static site generation capabilities for fast performance and easy deployment via GitHub Actions to GitHub Pages. The `hugo-book` theme is chosen for its simplicity, responsiveness, and support for multi-language content.

## Local Development

### Prerequisites

- [Hugo](https://gohugo.io/installation/) (Extended version recommended)
- Git

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/yhh2908307500/superking-blog.git
   cd superking-blog
   ```
2. Install dependencies (if any):
   ```bash
   npm install
   ```
3. Start the local server:
   ```bash
   hugo server
   ```
4. Open your browser and navigate to [http://localhost:1313](http://localhost:1313) to view the site.

## Deployment

### GitHub Pages

1. Configure the `config.toml` file with your GitHub Pages settings.
2. Push your changes to the `main` branch.
3. GitHub Actions will automatically build and deploy your site to GitHub Pages.
